Project Outline

My first The Odin Project project! In this project I will be building a basic recipe website using HTML only. It won't look pretty, but that's not the point. The purpose is to use the HTML knowledge and skills that I've acquired to this point:

   - Creating html files and adding html boilerplate
   - Validating markup to so that it complies with best practices
   - Basic heading, paragraph, and list structures and nesting
   - Using <a> tags and href attributes to create absolute and relative links
   - Nested folder structure for website
   - Using <img> tags and src attributes to add images
   
I'm probably missing a few things, but that's the gist. I will try to go a little deeper in terms and learn some more attributes to see what I can do with HTML only. 

At the conclusion of the project I will update this readme file with a reflection. Here we go!!
